# 终末地白色大字生成器网页版

## 简介

终末地白色大字生成器网页版，顾名思义就是用来生成终末地的白色大字的，而且是网页版

## 使用方法

访问 https://rspqfgn.github.io/EndfieldTextGenerator-web/

## 相关技术支持

原项目：https://github.com/RXLX-CHINA/EndfieldTextGenerator

萨卡兹字母字体：https://github.com/lhclbt/Endfield_Font

Pywubi库（原生的支持的是五笔86，但是终末地用的是五笔98，所以原作者更新了一下）
五笔98字库：http://98wb.ysepan.com/

## 注意事项

该项目使用AIGC工具移植，可能会出现史山代码和一些bug，请见谅
